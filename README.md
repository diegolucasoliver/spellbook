# SpellBook

O SpellBook é um aplicativo Android moderno para consulta de magias do sistema de RPG Tormenta 20, construído com as melhores práticas de desenvolvimento.

## Tecnologias Utilizadas

*   **Jetpack Compose:** Interface reativa e moderna.
*   **Firebase Firestore:** Banco de dados NoSQL em tempo real.
*   **Firebase Authentication:** Login anônimo para acesso seguro aos dados.
*   **MVVM & Coroutines:** Arquitetura limpa e processamento assíncrono.
*   **JUnit & MockK:** Suíte de testes unitários para regras de negócio.

## Estrutura do Projeto

As diretrizes de desenvolvimento para IAs e contribuidores estão localizadas na pasta `.ai/`.
